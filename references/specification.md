# 面经整理格式规范

## 文档结构

```
# Agent与大模型应用面经汇总

## 一、今日摘要
## 二、按公司分类整理
## 三、高频问题归纳
## 四、给 Bruno 的针对性建议
## 五、原始链接列表
```

---

## 一、今日摘要

输出以下统计信息：

- 今日新增帖子数
- 去重后有效帖子数
- 涉及公司数
- 高频岗位方向
- 今日最常见的 10 个面试主题
- 今日最值得关注的 3 条面经（附链接）

---

## 二、按公司分类整理

按公司分组，每个公司一个二级标题。公司名使用 `<Mark color="...">` 包裹，颜色映射如下：

| 公司 | 颜色 |
|------|------|
| 字节跳动 | red |
| 腾讯 | blue |
| 阿里/阿里云 | orange |
| 百度 | purple |
| 美团 | yellow |
| 快手 | purple |
| 小米 | green |
| 京东 | grey |
| OpenAI / Anthropic / 海外 AI | sky_blue |
| 其他公司 | default |

每个公司下用表格展示，字段包括：

| 字段 | 说明 |
|------|------|
| 公司 | 公司名（带颜色标记） |
| 岗位 | 岗位名称 |
| 是否实习/校招/社招 | 招聘类型 |
| 面试轮次 | 一面/二面/三面/HR面等 |
| 发布时间 | 帖子发布日期 |
| 作者 | 小红书作者昵称 |
| 原帖链接 | 带 xsec_token 的完整链接 |
| 面经摘要 | 提炼后的核心内容（非原文复制） |
| 高频问题 | 帖子中提到的面试问题 |
| 回答要点 | 作者分享的回答思路 |
| 对 Agent/大模型应用岗位准备的启发 | 对求职者的参考价值 |

---

## 三、高频问题归纳

汇总所有帖子中出现频率最高的问题，按以下主题分类：

- Agent 架构设计
- Prompt / Workflow / Skill / Tool Calling
- RAG / 检索 / 向量库 / 重排
- LangChain / LangGraph / 多 Agent
- 大模型基础
- 项目介绍与难点
- 工程化 / 部署 / 评估
- 八股 / 反问 / 行为面

每个问题下给出：

1. 问题本身
2. 常见考察点
3. 推荐回答思路（简洁版）

## 四、给 Bruno 的针对性建议

结合"Agent 开发岗位求职准备"场景，输出：

- 今天这些面经反映出的能力要求
- 最该补的知识点
- 面试中最值得准备的项目讲法
- 明天可以重点刷的 10 个问题

## 内容筛选规则

### 保留条件

- 真实面试经验、面试复盘、面试题总结、岗位经历分享
- 优先保留包含：公司名、岗位名、面试轮次、高频问题、回答思路、结果或反馈

### 过滤条件

- 广告、培训引流、付费社群推广
- 纯情绪帖、无实质内容的帖子
- spam

### 去重规则

- 标题相近、链接相同、内容高度重复的帖子只保留一条
- 优先保留信息更完整的版本

### 时间范围

- 搜索最近 30 天内的公开帖子
- 优先近 7 天

### 缺失信息处理

- 无法明确识别公司名时，标记为"其他/未明确"
- 信息不完整时用"未明确"标记，不要编造
